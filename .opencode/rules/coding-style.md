# Código Asociarg Mobile

Usar siempre:

- TypeScript
- Componentes funcionales
- Interfaces para Props
- Hooks separados
- Código modular
- Exportaciones mediante index.ts


Convenciones:

Archivos:

kebab-case

Ejemplo:

payment-card.tsx


Componentes:

PascalCase

Ejemplo:

PaymentCard


Hooks:

use-nombre

Ejemplo:

usePayments


Mantener funciones pequeñas.

Evitar archivos gigantes.

Separar UI, lógica y datos.
