# Design System Asociarg

Toda interfaz debe utilizar tokens del sistema.

Ubicación:

src/theme/


Nunca usar:

- colores hardcodeados
- tamaños arbitrarios
- sombras manuales


Usar:

colors
spacing
typography
radii
shadows


Componentes:

Atomo:
src/components/ui/


Organismo:
src/features/*/components/


Los componentes UI no tienen lógica de negocio.
