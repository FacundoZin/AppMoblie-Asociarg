# Feature Development Skill

1. Crear dentro de src/features.

2. Crear estructura:

components/
screens/
hooks/
services/
types/
index.ts


3. Separar:

UI
Lógica
Datos
Tipos


Antes de crear archivos:

explicar qué se creará y por qué.
