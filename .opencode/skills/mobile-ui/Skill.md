# Mobile UI Skill


Crear interfaces siguiendo:

- diseño mobile first
- espacios consistentes
- accesibilidad
- componentes reutilizables


Usar:

Manrope

Design tokens Asociarg


Priorizar:

- claridad
- jerarquía visual
- buen uso del espacio
- estados vacíos
- loading states
- error states


No crear pantallas genéricas sin diseño.
